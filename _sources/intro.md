# Predicción de enfermedad cardíaca mediante pipelines y MLOps

Este proyecto desarrolla un flujo completo de MLOps para la predicción de enfermedad cardíaca.

## Contenido

- Demostración de data leakage
- Pipeline con validación cruzada y GridSearchCV
- API con FastAPI
- Docker
- Kubernetes
- CI con GitHub Actions
- Monitoreo con Evidently
